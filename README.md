## Prerequirements

- Docker, docker-compose

## Get Started

### Develop

1. `docker-compose run app /bin/bash`

#### Test

```bash
yarn test
```

#### Linter, Formatter

-   Lint

```bash
yarn lint
```

-   Format

```bash
yarn format
```
